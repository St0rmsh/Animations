#  AI Battle Arena

> **Where AI models compete, and an AI judge decides the winner.**

AI Battle Arena is a modern web application that allows users to compare the performance of different Large Language Models (LLMs) on the same prompt. A single user query is sent to two competing AI models simultaneously, and a dedicated Judge AI evaluates their responses based on accuracy, reasoning, relevance, clarity, and completeness before declaring the winner.

The platform provides an interactive and transparent way to benchmark AI models, helping developers, researchers, and AI enthusiasts understand the strengths and weaknesses of different models across various tasks.

##  How It Works

1. The user enters a prompt.
2. The prompt is sent to two different AI models.
3. Both models generate their responses independently.
4. The Judge AI analyzes each response using multiple evaluation criteria:

   * Accuracy
   * Reasoning
   * Relevance
   * Clarity
   * Completeness
5. The Judge AI provides:

   * Individual scores
   * Strengths and weaknesses
   * A detailed explanation
   * The overall winner
